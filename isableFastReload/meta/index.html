How its made ?

This website is made using Hugo, a static site generator. The theme I use is based on Hugo Bearblog. Which in turn is based on Bear blog. I borrowed the color scheme from somestyleplease, another minimal monospace Hugo theme. You could call this a Hugo Bearblog/somestyleplease hybrid. I saw Rogs use the Archie theme and I liked it so I took inspiration from there for the Monospace/Sans-serif jugalbandi and the really neat header. Rogs in the neighbourhood somewhere in all those webrings. I also borrow some stuff from Hugo Theme Console, which I was using prior to this for about two years. The Gallery and Scrapbook use a layout based on Bill Glover’s Image Gallery. I was inspired to make a humans.txt file from Luke Davis

Some (excessive) notes about the color scheme and theme

The website at the moment has three themes, Dark 🌑, Bleach ☀️ (Light) and Puke 🤢 (Piss)

The original dark and light color schemes as explained before were borrowed from somestyleplease. Later, I’ve employed a theme switcher based on Sarah Fosshiem’s Theme Switcher. I was inspired to do this while dithering a picture of myself for the front page (for asthetics and space savings and maybe privacy even). I liked the different color pallets of ditherit.com (not to be confused with dither.it another cool dithering website btw). I thought to myself, what if the profile image switches with the theme that would be cool !? Eventually the Gameboy theme was made, eventually the gameboy theme was named piss (Even though it was too green for that, House MD?) Eventually, I renamed piss to puke, I like piss puke, piss puke is my favourite theme. Also, I’ve modified the Light theme and renamed it to Bleach, because why not it feels like pouring bleach into my eyes. Well, not so much since I changed the background color from #ffffff to #bcbcbc which complements the fact that this is a website written in Delhi. But still, given the toxic nature of bleach I’ve taken to wearing some PPE whilist you make me handle the chemicals. Also dont mind while I have my institutional snack of tea in a microplastic laced papercup while I am not staring at you from the backseat. Bleach in my opinion was the best album Nirvana made (not really).

Drink bleach, Puke🤢.
